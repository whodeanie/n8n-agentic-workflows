# Revenue Pipeline Coach

## Pattern

Scheduled deal review, pipeline scoring, next-best-action generation, risk routing, CRM task creation, and leadership digest.

## Flow

Pipeline Review Schedule -> Fetch Open Deals -> Expand Deals -> Deal Coach Agent -> Segment Deal -> Create Rep Task or Risk Alert -> Audit -> Digest.

## Useful For

Sales operations, customer success handoffs, renewal tracking, account planning, and revenue teams that need disciplined follow-up without replacing manager judgment.

## Setup Notes

Reconnect HubSpot or Salesforce, OpenAI, Slack, and Google Sheets credentials. Replace the pipeline query, task fields, rep channel, and leadership digest destinations before live use.

## Validation Focus

The agent creates recommendations and evidence-backed tasks, while the workflow keeps risky deals visible through deterministic routing and audit rows.
