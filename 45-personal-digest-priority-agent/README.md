# Personal Digest Priority Agent

## Pattern

Inbox, calendar, tasks, and saved-link aggregation, priority scoring, action extraction, digest drafting, and review audit.

## Flow

Daily Digest Schedule -> Fetch Inbox Signals -> Fetch Calendar -> Fetch Tasks -> Merge Daily Signals -> Digest Agent -> Urgency Gate -> Priority Alert or Morning Digest -> Audit.

## Useful For

Busy weeks, job searching, personal admin, side projects, family coordination, creator planning, and anyone who wants one calm daily brief instead of twelve tabs.

## Setup Notes

Reconnect OpenAI, Gmail, Google Calendar, Slack, and Google Sheets credentials. Replace mailbox labels, calendar ID, task source, digest channel, and audit sheet.

## Validation Focus

The workflow separates urgent actions from informational items, preserves source references, and routes high-risk or time-sensitive tasks as alerts instead of burying them in a digest.
